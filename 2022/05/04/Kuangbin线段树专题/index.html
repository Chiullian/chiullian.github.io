AcWing敌兵布阵(单点修改区间查询和)

敌人有NNN 个工兵营地，编号1∼N1∼N1∼N。

初始时，第iii 个营地有aia_iai​ 个人。

接下来有若干个命令，命令有 44 种形式：

  • Add i j，iii 和jjj 为正整数，表示第iii 个营地增加jjj 个人。（jjj不超过303030）
  • Sub i j，iii 和jjj 为正整数，表示第iii 个营地减少jjj 个人。（jjj不超过303030）
  • Query i j，iii 和jjj 为正整数（i≤ji≤ji≤j），表示询问第iii 到第jjj 个营地的总人数。
  • End，表示结束，此命令只会作为最后一条命令出现。

请你计算每个 Query 的答案。

板子不多说

AcWing 4340. 我讨厌它(单点修改区间查询max)

很多学校流行一种比较的习惯。

老师们很喜欢询问，从某某到某某当中，分数最高的是多少。

这让很多学生很反感。

不管你喜不喜欢，现在需要你做的是，就是按照老师的要求，写一个程序，模拟老师的询问。

当然，老师有时候需要更新某位同学的成绩。

输入格式

本题目包含多组测试，请处理到文件结束。

在每个测试的第一行，有两个正整数NNN 和MMM，分别代表学生的数目和操作的数目。

学生 ID 编号分别从111 编到NNN。

第二行包含NNN 个整数，代表这NNN 个学生的初始成绩，其中第iii 个数代表 ID 为iii 的学生的成绩。

接下来有MMM 行。每一行有一个字符CCC（只取 Q 或 U），和两个正整数A,BA,BA,B。

当CCC 为 Q 的时候，表示这是一条询问操作，它询问 ID 从AAA 到BBB（包括A,BA,BA,B）的学生当中，成绩最高的是多少。

当CCC 为 U 的时候，表示这是一条更新操作，要求把 ID 为AAA 的学生的成绩更改为BBB。

输出格式
对于每一次询问操作，在一行里面输出最高成绩。

pushup的时候按照最大值即可

AcWing 243. 一个简单的整数问题2(区间修改+,区间查询和)

给定一个长度为 N 的数列 A，以及 M 条指令，每条指令可能是以下两种之一：

  1. C l r d，表示把 A[l],A[l+1],…,A[r] 都加上 d。
  2. Q l r，表示询问数列中第 l∼r 个数的和。

对于每个询问，输出一个整数表示答案。

将线段树的节点的懒标记下放到儿子上，根节点的懒标记清空

分为：产生影响+懒标记下放

产生的影响是儿子区间内所有的数的总和增加了区间的长度×增加值区间的长度×增加值区间的长度×增加值

根的懒标记清空

AcWing 4341. 市长海报(离散化+区间修改成某个数字查询数字种类个数)

在一个市长选举宣传墙上，有nnn 张海报正等待着秘书依次张贴。

所有海报都与墙面等高。

整个墙面被等分为若干段，从左到右依次编号为1,2,3…1,2,3…1,2,3…。

每张海报都恰好完全覆盖连续若干段墙面，其中第 ii 张海报覆盖第li∼ril_i∼r_ili​∼ri​段墙面。

当所有海报都张贴完毕以后，其中一些海报可能会被后面的海报覆盖掉。

请你计算，最终一共有多少张海报（完全或部分）可见。

输入格式

第一行包含整数TTT，表示共有TTT 组测试数据。

每组数据第一行包含整数nnn。

接下来nnn 行，每行包含两个整数li,ril_i,r_ili​,ri​，表示一张海报的覆盖区域。

输出格式

每组数据输出一行结果，一个整数，表示最终可见的海报数量。

思路:

每次把修改的区间标记成当前的编号,看最后区间内还剩下多少个编号种类可以看见,数据范围太大,而且数据量不大,离散化即可.

这里还有一个特殊的情况就是处理一号232323区间,二号45区间和三号252525区间,按照正常的区间修改,最后是能看见第一个和第二个区间,然而事实上第三个区间也能看到,所以我们可以把r+1r + 1r+1也进行离散化及分别对l,r,r+1l,r,r+1l,r,r+1进行离散化,利用r+1r+1r+1判断是否是这种情况

这里的pushdown操作解释:

产生影响+懒标记下放

产生的影响

把儿子节点的编号全部染成父节点的编号,然后在给儿子标记再往下递归

AcWing 4342. 就一勾子(区间修改数字,区间查询和)

在 DotA 这个游戏中，帕吉（屠夫、胖子）的钩子让很多英雄闻风丧胆。

帕吉的钩子是由 nn 个连续的等长金属棒拼接而成。

这些金属棒不妨编号为1∼n1∼n1∼n。

最开始时，所有金属棒都是铜棒。

现在，帕吉要对他的钩子进行QQQ 次更改操作，操作分为以下三种：

  • l r 1，表示将第l∼rl∼rl∼r 个金属棒换成铜棒。
  • l r 2，表示将第l∼rl∼rl∼r 个金属棒换成银棒。
  • l r 3，表示将第l∼rl∼rl∼r 个金属棒换成金棒。

每根铜棒的价值为111，每根银棒的价值为222，每根金棒的价值为333。

钩子的总价值等于nnn 个金属棒的价值之和。

请你计算所有操作完成以后，钩子的总价值。

pushdown操作解释

分为：产生影响+懒标记下放

产生的影响是 维护的区间和就是区间的长度×修改的价值区间的长度×修改的价值区间的长度×修改的价值

AcWing 4343. 数颜色(区间修改)

我们要在一个数轴上绘制nnn 个彩色线段。

先绘制的线段可能会被后绘制的线段覆盖。

你的任务是统计最终可以看到的不同颜色的线段。

输入格式

输入包含多组测试数据。

每组数据第一行包含整数nnn。

接下来nnn 行，每行包含三个整数l,r,cl,r,cl,r,c，表示绘制左右端点坐标分别为l,rl,rl,r 的颜色为ccc 的线段。

思路

思路:

根据此图我们可以看到一个小技巧就是把做区间l+1l + 1l+1

这样区间0−40-40−4就可以表示成第1,2,3,4段,然后在进行区间修改查询即可

每天，农夫 John 的NNN 头牛总是按同一序列排队。

有一天，John 决定让一些牛玩一场飞盘比赛。

他准备找一群在队列中位置连续的牛来进行比赛，但是为了避免水平悬殊，牛的身高不应该相差太大。

John 准备了QQQ 个可能的牛的选择和所有牛的身高。

他想知道每一组里面最高和最矮的牛的身高差别。
