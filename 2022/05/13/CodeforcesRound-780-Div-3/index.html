Vasya and Coins

Vasya decided to go to the grocery store. He found in his wallet aaa coins of 111 burle and bbb coins of 222 burles. He does not yet know the total cost of all goods, so help him find out sss (s>0s > 0s>0): the minimum positive integer amount of money he cannot pay without change or pay at all using only his coins.For example, if a=1a=1a=1 and b=1b=1b=1 (he has one 111-burle coin and one 222-burle coin), then:

  • he can pay 111 burle without change, paying with one 111-burle coin,
  • he can pay 222 burle without change, paying with one 222-burle coin,
  • he can pay 333 burle without change by paying with one 111-burle coin and one 222-burle coin,
  • he cannot pay 444 burle without change (moreover, he cannot pay this amount at all).

So for a=1a=1a=1 and b=1b=1b=1 the answer is s=4s=4s=4.

题意 你有a枚一元硬币和b枚2元硬币，问你最小不能组合的零钱数

没有1元就拼不出1元等奇数, 有的话能拼到的最大的是a + 2 * b

Vlad and Candies

Not so long ago, Vlad had a birthday, for which he was presented with a package of candies. There were nnn types of candies, there are aia_iai​ candies of the type iii (1≤i≤n1 \le i \le n1≤i≤n).Vlad decided to eat exactly one candy every time, choosing any of the candies of a type that is currently the most frequent (if there are several such types, he can choose any of them). To get the maximum pleasure from eating, Vlad does not want to eat two candies of the same type in a row.Help him figure out if he can eat all the candies without eating two identical candies in a row.

收到一包含有n种类的糖果，你每次不能连续两次吃相同种类的糖果并且只能吃数量最多的糖果，问你能不能把所有糖吃完,

思路:我们可以马上发现,只需要最多的糖果与第二多的糖果之间的数量不超过2,就能一直往下吃,碰见小的就会有更多的选择

Get an Even String

A string a=a1a2…ana=a_1a_2\dots a_na=a1​a2​…an​ is called even if it consists of a concatenation (joining) of strings of length 222 consisting of the same characters. In other words, a string aaa is even if two conditions are satisfied at the same time:

  • its length nnn is even;
  • for all odd iii (1≤i≤n−11 \le i \le n - 11≤i≤n−1), ai=ai+1a_i = a_{i+1}ai​=ai+1​ is satisfied.

For example, the following strings are even: “” (empty string), “tt”, “aabb”, “oooo”, and “ttrrrroouuuuuuuukk”. The following strings are not even: “aaa”, “abab” and “abba”.Given a string sss consisting of lowercase Latin letters. Find the minimum number of characters to remove from the string sss to make it even. The deleted characters do not have to be consecutive.

题意:

定义一个字符串是合法的满足 1. 长度为偶数 2. 所有奇数位的字符都等于下一个字符
例如“aabb”，“ttllqq”是合法的,给你你个字符串，问你最少删除多少个字串能令其变成合法字符串

思路:

如果直接求删除的话很难维护删除后的信息, 所以想到求最多能组成一对一对的的长度,这里有两种思路,一种是dp,还有一种是贪心,

  1. DPDPDP, 状态转移方程的含义fif_ifi​ 前i个字母中能拼成两两一对的最长的(可以不连续), 下面考虑如何转移, 如果,开一个数组来记录,当前字母前面出现的位置, 转移的上一个位置从上一个出现相同字母的上一个位置转移

  2. 贪心贪心贪心, 如果字母之前出现过就加一对字母ans+2然后清空标记

Maximum Product Strikes Back

You are given an array aaa consisting of nnn integers. For each iii (1≤i≤n1 \le i \le n1≤i≤n) the following inequality is true: −2≤ai≤2-2 \le a_i \le 2−2≤ai​≤2.

You can remove any number (possibly 000) of elements from the beginning of the array and any number (possibly 000) of elements from the end of the array.

You are allowed to delete the whole array.You need to answer the question: how many elements should be removed from the beginning of the array, and how many elements should be removed from the end of the array, so that the result will be an array whose product (multiplication) of elements is maximal. If there is more than one way to get an array with the maximum product of elements on it, you are allowed to output any of them. The product of elements of an empty array (array of length 000) should be assumed to be 111.

题意

给你一个元素大小绝对值小于等于2的数组，你可以从头和从尾部删去若干个数，问你怎么删去原色能使剩下元素中乘积最大，如果数组为空，乘积为 1

思路

其实这题思路好想但是边界处理比较恶心,算是锻炼一下代码能力了吧

答案肯定不是0,题目说空是1, 所有0是划分线, 所以只需要看所有分块的最大值即可,只有2有贡献值所有统计2的个数即可,因为值会很大所以统计2的个数不能直接存,如果分块内的负数数量是偶数个,直接进行比即可,如果是奇数,枚举从左边开始减直到减到第一个负数,同理从右边枚举,比较最大的即可,还是有很多细节的,见代码

Matrix and Shifts

You are given a binary matrix AAA of size n×nn \times nn×n. Rows are numbered from top to bottom from 111 to nnn, columns are numbered from left to right from 111 to nnn. The element located at the intersection of row iii and column jjj is called AijA_{ij}Aij​. Consider a set of 444 operations:

  1. Cyclically shift all rows up. The row with index iii will be written in place of the row i−1i-1i−1 (2≤i≤n2 \le i \le n2≤i≤n), the row with index 111 will be written in place of the row nnn.
  2. Cyclically shift all rows down. The row with index iii will be written in place of the row i+1i+1i+1 (1≤i≤n−11 \le i \le n - 11≤i≤n−1), the row with index nnn will be written in place of the row
  3. Cyclically shift all columns to the left. The column with index jjj will be written in place of the column j−1j-1j−1 (2≤j≤n2 \le j \le n2≤j≤n), the column with index 111 will be written in place of the column nnn.
  4. Cyclically shift all columns to the right. The column with index jjj will be written in place of the column j+1j+1j+1 (1≤j≤n−11 \le j \le n - 11≤j≤n−1), the column with index nnn will be written in place of the column 111.

You can perform an arbitrary (possibly zero) number of operations on the matrix; the operations can be performed in any order.After that, you can perform an arbitrary (possibly zero) number of new xor-operations:

  • Select any element AijA_{ij}Aij​ and assign it with new value Aij⊕1A_{ij} \oplus 1Aij​⊕1. In other words, the value of (Aij+1) mod 2(A_{ij} + 1) \bmod 2(Aij​+1)mod2 will have to be written into element AijA_{ij}Aij​.

Each application of this xor-operation costs one burl. Note that the 444 shift operations — are free. These 444 operations can only be performed before xor-operations are performed.Output the minimum number of burles you would have to pay to make the AAA matrix unitary. A unitary matrix is a matrix with ones on the main diagonal and the rest of its elements are zeros (that is, Aij=1A_{ij} = 1Aij​=1 if i=ji = ji=j and Aij=0A_{ij} = 0Aij​=0 otherwise).

题意

每次可以对几个元素^1就是变换一下0变1, 1变0;问最少多少次可以是只有主对角线上的值为1,可以循环移动每一行和每一列,循环不花费,改变花费

直接计算每一条主对角线上和上下循环后能成为主对角线上的点,出现1的数量最多的为把其他的多余的1都去掉,然后同一对角线为0的变为1

Promising String (hard version)

This is the easy version of Problem F. The only difference between the easy version and the hard version is the constraints.We will call a non-empty string balanced if it contains the same number of plus and minus signs. For example: strings “±-+” and “+±±-” are balanced, and strings “±-”, “–” and “” are not balanced.We will call a string promising if the string can be made balanced by several (possibly zero) uses of the following operation:

  • replace two adjacent minus signs with one plus sign.

In particular, every balanced string is promising. However, the converse is not true: not every promising string is balanced.For example, the string “-±–” is promising, because you can replace two adjacent minuses with plus and get a balanced string “-+±”, or get another balanced string “-±+”.How many non-empty substrings of the given string sss are promising? Each non-empty promising substring must be counted in the answer as many times as it occurs in string sss.Recall that a substring is a sequence of consecutive characters of the string. For example, for string “±+” its substring are: “±”, “-+”, “+”, “±+” (the string is a substring of itself) and some others. But the following strings are not its substring: “–”, “++”, “-++”.

F1和F2题意一样，区别为F1的n为1≤n≤3000，F2为1≤n≤2e5,给你只含“+”，“-”的字符串，你可以把相邻两个“-”转化为“+” ， 问有多少个子字符串可以通过这种操作使“+”数量等于“-”，例如“±”，“±–”，都是合法的

因此用前缀和记录符号数量遇-就加1否则减1

就变成了多少连续子段和大于零

要完成上述的条件需要满足的条件

  1. -号数量必须大于等于+号数量
  2. -号数量减去 +数量必须是3的整数倍(每两个-都可以变成一个+来平衡,就是花费3个)

第一种方法直接暴力枚举区间的左端点和右端点O(n2n^2n2)

第二种(s[r] - s[l - 1]) % 3 == 0 就是 s[r ] = s[l] 在模3的意义下

且 s[l - 1] ≤\leq≤ s[r]

所以在同一个模数的情况下求小于 s[r]的数有多少就行,然后就是树状数组,带3个偏序的树状数组,分别代表模为0, 模为1, 模为2的情况下的树状数组, 还得加一个偏移量,因为s[i]有可能是小于零的数
