[Palindromic Indices]

You are given a palindromic string sss of length nnn.You have to count the number of indices iii (1≤i≤n)(1 \le i \le n)(1≤i≤n) such that the string after removing sis_isi​ from sss still remains a palindrome. For example, consider sss = “aba” If we remove s1s_1s1​ from sss, the string becomes “ba” which is not a palindrome. If we remove s2s_2s2​ from sss, the string becomes “aa” which is a palindrome. If we remove s3s_3s3​ from sss, the string becomes “ab” which is not a palindrome. A palindrome is a string that reads the same backward as forward. For example, “abba”, “a”, “fef” are palindromes whereas “codeforces”, “acd”, “xy” are not.

题意：给定一个回文串s，问有多少个位置i，删除s[i]后字符串依旧是回文串？

只需要看中间那一串连续的可以删多少以可，即与中间的答案相同的有几个， 奇数的话答案加1

[AND Sorting]

You are given a permutation ppp of integers from 000 to n−1n-1n−1 (each of them occurs exactly once). Initially, the permutation is not sorted (that is, pi>pi+1p_i>p_{i+1}pi​>pi+1​ for at least one 1≤i≤n−11 \le i \le n - 11≤i≤n−1). The permutation is called XXX-sortable for some non-negative integer XXX if it is possible to sort the permutation by performing the operation below some finite number of times: Choose two indices iii and jjj (1≤i<j≤n)(1 \le i \lt j \le n)(1≤i<j≤n) such that pi&pj=Xp_i \& p_j = Xpi​&pj​=X. Swap pip_ipi​ and pjp_jpj​. Here &\&& denotes the bitwise AND operation.Find the maximum value of XXX such that ppp is XXX-sortable. It can be shown that there always exists some value of XXX such that ppp is XXX-sortable.

给你一个未排好序的的排列

你可以选择一个数

对于如果aia_iai​ & aja_jaj​ = xxx就可以进行交换

要求你将这个排列排好序同时让你找到最大的值xxx并输出

只需要看没有归位的即a[i]a[i]a[i] ≠\neq= iii的位置即可, 那么先把所有的不在相应位置上的&在一起再用与值去交换,不过不是这样的话就不能全部交换考虑 运算的一个特性，一个数进行 运算的话一定是越来越小的，当一个数是 “X-Sortable” 以后，我们再用 去并其他未排序的数，得到新的 ，这里的 是兼容 的，也即 “X-sortable” 的数一定 “X’-sortable”。

比如 0 3 4 1 2

3 4 1 2需要交换，与值为 3 & 4 & 1 & 2 = 0

我们通过0来进行交换

第一次交换为4 3 0 1 2

[LIS or Reverse LIS?]

​ You are given an array aaa of nnn positive integers. Let LIS(a)\text{LIS}(a)LIS(a) denote the length of longest strictly increasing subsequence of aaa. For example,

  • LIS([2,1‾,1,3‾])\text{LIS}([2, \underline{1}, 1, \underline{3}])LIS([2,1​,1,3​]) = 222.
  • LIS([3‾,5‾,10‾,20‾])\text{LIS}([\underline{3}, \underline{5}, \underline{10}, \underline{20}])LIS([3​,5​,10​,20​]) = 444.
  • LIS([3,1‾,2‾,4‾])\text{LIS}([3, \underline{1}, \underline{2}, \underline{4}])LIS([3,1​,2​,4​]) = 333.

​ We define array a′a'a′ as the array obtained after reversing the array aaa i.e. a′=[an,an−1,…,a1]a' = [a_n, a_{n-1}, \ldots , a_1]a′=[an​,an−1​,…,a1​].The beauty of array aaa is defined as min(LIS(a),LIS(a′))min(\text{LIS}(a),\text{LIS}(a'))min(LIS(a),LIS(a′)).Your task is to determine the maximum possible beauty of the array aaa if you can rearrange the array aaa arbitrarily.

给你一个数组

你可以任意排列这个数组

让你得到min(LIS(a),LIS(a′))min(\text{LIS}(a),\text{LIS}(a'))min(LIS(a),LIS(a′))的最大值并输出最大值

当有两个一样的时候我们都可以选, 当只有一个的时候,我们可以分成两部分,左边上升的与右边上升的,平均左边+1右边在+1这样就能是最后的答案最小的最大

[Circular Spanning Tree]

​ There are nnn nodes arranged in a circle numbered from 111 to nnn in the clockwise order. You are also given a binary string sss of length nnn.Your task is to construct a tree on the given nnn nodes satisfying the two conditions below or report that there such tree does not exist: For each node iii (1≤i≤n)(1 \le i \le n)(1≤i≤n), the degree of node is even if si=0s_i = 0si​=0 and odd if si=1s_i = 1si​=1. No two edges of the tree intersect internally in the circle. The edges are allowed to intersect on the circumference. Note that all edges are drawn as straight line segments. For example, edge (u,v)(u, v)(u,v) in the tree is drawn as a line segment connecting uuu and vvv on the circle.A tree on nnn nodes is a connected graph with n−1n - 1n−1 edges.

题意：

给你n个节点

并给出每个节点的度的奇偶性

问你能不能构造出符合要求的树

可以则给出构造方案

先看是否可行,因为肯定要有子节点,所以一定有一个是奇数的节点,树的性质:所有度的和一定是偶数,一条边贡献两个度, 不满足上面就是NO

如何构造: 把所有的为1的当做一个子节点, 然后把后面的0加入到链中,这样就能满足所有的子节点都是奇数, 然后把根节点合并即可.
