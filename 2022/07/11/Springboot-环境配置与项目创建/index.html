1.Git的配置

Git 工具用来 管理 本地仓库 和 云仓库(可以返回之前的任意一个版本)

下载地址

先创建项目文件夹,然后右键git bash 在框里输入git init初始化git仓库

然后:

记得把-- global去掉,因为加上就是全部使用该用户

  • git status表示当前目录下有多少文件还没有收录到仓库里
  • git add . 该文件夹下未收录的全部收录
  • git commit -m "提交的信息"(将暂存区提交到本地)
  • git push 上传到云端仓库

2.然后是创建项目

bootstrap前端常用的网站样式

SpringBoot中的常用模块

  • pojo层：将数据库中的表对应成Java中的Class
  • mapper层（也叫Dao层）：将pojo层的class中的操作，映射成sql语句
  • service层：写具体的业务逻辑，组合使用mapper中的操作
  • serviceimpl层: 这个是service接口的实现类,给controller层使用
  • controller层：负责请求转发，接受页面过来的参数，传给Service处理，接到返回值，再传给页面
