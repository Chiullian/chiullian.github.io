A.炸雷

2022年10月21日，雷俊手下的cm从睡梦中惊醒，发现手里出现一张猫猫大帝颁发的毁灭世界许可，敌视世界的cm刚好想趁此机会毁灭宇宙。

cm所在宇宙内的所有世界都是圆环形状，世界被均匀的分为n段，cm的好朋友lyf刚好在众客袁研究灭世武器，每个灭世武器爆炸最多可以毁灭连续的3段世界，cm想知道对于某个圆环世界，他应该找lyf借多少个灭世武器，才能完全毁灭这个圆环世界呢？

B.呆呆白羊小姐

兔子先生前几天因为淋雨生病了，白羊小姐打算去探望他。

白羊小姐打算给兔子先生买他最喜欢的水果 nn 千克。

水果店的水果每千克101010 元，每买555 千克水果送111 千克。

呆呆的白羊小姐不知道怎么计算价格，你可以帮助她嘛？QAQ

C.Peach_01‘s favourite

第一行给出n表示长度，随后两行分别给出长n的数组A和B

请试着构造一个长n的C数组，使得每个Ci = Ai = Bi，求min(mex©)

关于mex*:mex(S)的值为集合S中没有出现过的最小自然数。例如，mex({1,2}) = 0、mex({0,1,2,3}) = 4*

mex的含义。

输出min(mex©)

对于所有 A[i] == B[i]，被确定的数一定要避开，标记这些数并暂定一个

对于所有A[i] != B[i], 都能避开(猜的)

D.Socktail’s Scholarship

大佬们申请某奖学金，有评比规则如下：

评比均不关心绩点和加分项级别(一二三等)，只关心加分项名称和数量，且权重如下：

创新创业比赛：9分/个

软件著作权或发明专利：0.9分/个

学科竞赛得奖：0.1分/个

各大佬的评比得分为自己的Σ加分项数量*加分项权重Σ加分项数量∗加分项权重

请你依据得分进行排名，从高到低输出大佬的姓名

第一行给出两个整数n，按顺序依次表示大佬们提交的申请

随后n行每行一个由小写字母构成的字符串和四个整数：name名字(唯一，也就是说每个名字只出现一次), cy创奖数量, rz软著数量, zl专利数量, js竞赛得奖数量

1 <= n <= 1e6

1 <= len(name) <= 30

0 <= cy, rz, zl, js <= 1e15

E.笨笨兔子先生

流浪的兔子先生在蓝星遇见了很多朋友，每次兔子先生到朋友家做客，朋友们都会给兔子先生做一大桌子好吃的。

由于蓝星的朋友们都非常喜欢质数，因此饭桌上盘子背后都有一个质数作为标识。

朋友们每次都会在饭后告诉兔子先生，这顿饭所有盘子背后数字的乘积。

笨笨的兔子先生想知道哪些盘子是他所有朋友都共同拥有的，你能帮帮他嘛？QAQ

1≤n≤1e6，2≤ai≤1∗e∗121≤n≤1e6，2≤a_i≤1*e*121≤n≤1e6，2≤ai​≤1∗e∗12

有数的 gcd 就是所有朋友都有的盘子的乘积，然后进行朴素是质因数分解即可。

F.漫天繁星

自从2022年10月21日cm毁灭宇宙之后，蓝星又进入了新一轮演化。某一天，蓝星星主突然出现一段往日的回忆，想起了往日的恋人：月星。众所周知，四边形代表爱情[1]^{[1]}[1]，蓝星为了重新追求月星，准备计算出宇宙中有多少种[3]^{[3]}[3]四边形星座[2]^{[2]}[2]。

[1][1]: 作者胡扯罢了

[2][2]: 四边形星座是指利用四颗星星，每颗作为四边形的顶点。

[3][3]: 四边形星座中，存在一个与其它星座不同的星星，即为不同的四边形星座

个点能够组成一个四边形的条件是任意三个点不同线。 然后我们可以枚举选择哪四个点作为四边形的顶点，然后判断是否存在三点共线的非法情况即可
