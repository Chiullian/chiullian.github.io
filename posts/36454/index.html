[Array Decrements]

​ Kristina has two arraysaaa andbbb, each containingnnn non-negative integers. She can perform the following operation on arrayaaa any number of times: apply a decrement to each non-zero element of the array, that is, replace the value of each elementaia_iai​ such thatai>0a_i > 0ai​>0 with the valueai−1a_i - 1ai​−1 (1≤i≤n1 \le i \le n1≤i≤n). Ifaia_iai​ was000, its value does not change. Determine whether Kristina can get an arraybbb from an arrayaaa in some number of operations (probably zero). In other words, can she makeai=bia_i = b_iai​=bi​ after some number of operations for each1≤i≤n1 \le i \le n1≤i≤n?For example, letn=4n = 4n=4,a=[3,5,4,1]a = [3, 5, 4, 1]a=[3,5,4,1] andb=[1,3,2,0]b = [1, 3, 2, 0]b=[1,3,2,0]. In this case, she can apply the operation twice: after the first application of the operation she getsa=[2,4,3,0]a = [2, 4, 3, 0]a=[2,4,3,0]; after the second use of the operation she getsa=[1,3,2,0]a = [1, 3, 2, 0]a=[1,3,2,0]. Thus, in two operations, she can get an arraybbb from an arrayaaa.

[Restoring the Duration of Tasks]

题意：n个任务，有初始时间和结束时间，求任务的持续时间，任务的执行按照顺序执行，如果当前执行a任务，但是b任务已经到底初始时间，那么b任务等待a任务执行完之和开始，依次类推（可能会有多个任务在等待，按照顺序等着执行即可）

模拟过程即可,记录经历过的最大的bi

[Black and White Stripe]

题意：一个字符串有B和W两种字符，代表黑和白，现在可以进行操作，把白色染色成为黑色，问得到k个连续的黑色最少需要操作多少次

思路:

  1. 用前缀和即可,看有B最多的区间距为k的

  2. 滑动窗口

[Price Maximization]

题意：n（偶数）个商品，现在每两个打包在一起，他们的成本为价格总和除k，问成本最大

思路:

每个商品除k就是基本的成本,然后看取余k的存在数组中,然后贪心余数最小的加余数最大的是否>= k 大于的话贡献在加1

[Shifting String]

题意:给你一个字符串数组,再给你一个变换排列(按照对应的位置进行变换),看最少进行多少次排列的变换能回到原来的字符串数组

思路:

经过一定数量的操作后，每个字符都会回到它的位置。把每一个循环看作是一个每次循环移动的弦。答案是 lcm(最小公倍数)所有的周期长度, 最小循环节kmp判断即可
