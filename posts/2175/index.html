He does not yet know the total cost of all goods, so help him find outsss (s>0s > 0s>0): the minimum positive integer amount of money he cannot pay without change or pay at all using only his coins.For example, ifa=1a=1a=1 andb=1b=1b=1 (he has one111-burle coin and one222-burle coin), then:

Get an Even String

A stringa=a1a2…ana=a_1a_2\dots a_na=a1​a2​…an​ is called even if it consists of a concatenation (joining) of strings of length222 consisting of the same characters. In other words, a stringaaa is even if two conditions are satisfied at the same time:

题意:

思路:

  1. DPDPDP, 状态转移方程的含义fif_ifi​ 前i个字母中能拼成两两一对的最长的(可以不连续), 下面考虑如何转移, 如果,开一个数组来记录,当前字母前面出现的位置, 转移的上一个位置从上一个出现相同字母的上一个位置转移

  2. 贪心贪心贪心, 如果字母之前出现过就加一对字母ans+2然后清空标记

Maximum Product Strikes Back

You are given an arrayaaa consisting ofnnn integers. For eachiii (1≤i≤n1 \le i \le n1≤i≤n) the following inequality is true:−2≤ai≤2-2 \le a_i \le 2−2≤ai​≤2.

If there is more than one way to get an array with the maximum product of elements on it, you are allowed to output any of them. The product of elements of an empty array (array of length000) should be assumed to be111.

题意

给你一个元素大小绝对值小于等于2的数组，你可以从头和从尾部删去若干个数，问你怎么删去原色能使剩下元素中乘积最大，如果数组为空，乘积为 1

思路

其实这题思路好想但是边界处理比较恶心,算是锻炼一下代码能力了吧

答案肯定不是0,题目说空是1, 所有0是划分线, 所以只需要看所有分块的最大值即可,只有2有贡献值所有统计2的个数即可,因为值会很大所以统计2的个数不能直接存,如果分块内的负数数量是偶数个,直接进行比即可,如果是奇数,枚举从左边开始减直到减到第一个负数,同理从右边枚举,比较最大的即可,还是有很多细节的,见代码

Promising String (hard version)

This is the easy version of Problem F. The only difference between the easy version and the hard version is the constraints.We will call a non-empty string balanced if it contains the same number of plus and minus signs. For example: strings “±-+” and “+±±-” are balanced, and strings “±-”, “–” and “” are not balanced.We will call a string promising if the string can be made balanced by several (possibly zero) uses of the following operation:

  • replace two adjacent minus signs with one plus sign.

In particular, every balanced string is promising. However, the converse is not true: not every promising string is balanced.For example, the string “-±–” is promising, because you can replace two adjacent minuses with plus and get a balanced string “-+±”, or get another balanced string “-±+”.How many non-empty substrings of the given stringsss are promising? Each non-empty promising substring must be counted in the answer as many times as it occurs in stringsss.Recall that a substring is a sequence of consecutive characters of the string. For example, for string “±+” its substring are: “±”, “-+”, “+”, “±+” (the string is a substring of itself) and some others. But the following strings are not its substring: “–”, “++”, “-++”.

F1和F2题意一样，区别为F1的n为1≤n≤3000，F2为1≤n≤2e5,给你只含“+”，“-”的字符串，你可以把相邻两个“-”转化为“+” ， 问有多少个子字符串可以通过这种操作使“+”数量等于“-”，例如“±”，“±–”，都是合法的

因此用前缀和记录符号数量遇-就加1否则减1

就变成了多少连续子段和大于零

要完成上述的条件需要满足的条件

  1. -号数量必须大于等于+号数量
  2. -号数量减去 +数量必须是3的整数倍(每两个-都可以变成一个+来平衡,就是花费3个)

第一种方法直接暴力枚举区间的左端点和右端点O(n2n^2n2)

第二种(s[r] - s[l - 1]) % 3 == 0 就是 s[r ] = s[l] 在模3的意义下

且 s[l - 1]≤\leq≤ s[r]

所以在同一个模数的情况下求小于 s[r]的数有多少就行,然后就是树状数组,带3个偏序的树状数组,分别代表模为0, 模为1, 模为2的情况下的树状数组, 还得加一个偏移量,因为s[i]有可能是小于零的数
